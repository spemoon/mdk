array.indexOf

说明

数组中检索元素，找到返回该元素的下标，没找到返回-1

用法 
                    /**
                     * 查找数组中是否存在某个值
                     * @param {Mix} v 被搜索的值
                     * @param {Array} arr 被查找的数组
                     * @return {Number} 第一个匹配的元素的下标，没找到返回-1
                    */
                    array.indexOf(value, arr);
示例 (点击查看源码)

数组:[1,3,4,6,6,8,9]

array.forEach

说明

数组中遍历元素，每个元素执行一个函数

用法 
                    /**
                     * 遍历一个数组，每个元素执行callback，callback作用域是scope
                     * @param {Function} callback 执行函数，参数是 value，index，array
                     * @param {Array} arr 数据源
                     * @param {Object} scope 执行函数的作用域
                     * @return {undefined}
                     */
                    array.forEach(callback, arr, scope);
示例 (点击查看源码)

数组:[1,3,4,6,6,8,9]
